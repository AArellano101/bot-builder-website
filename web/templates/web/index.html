Failsafe



While the bot is running, the failsafe is activated. If the failsafe password is typed by the user, the bot stops running. The failsafe screen shows what characters have been typed by the user. The failsafe password allows the user to customize the password.

Note: While bot is running, password must be typed slowly, at half-second intervals.
Save & Run



After customizing your wanted bot, save it by pressing the save button. Doing so will post the bot into the explore page and will redirect you into the explore page. Although, while making the bot, the user can test the bots actions by pressing the Run button, which will run the actions in the action list from top to bottom.
Coordinate Tracker



Turning on the coordinate tracker will allow for the user to see the coordinates of the mouse on the screen. Use the coordinate tracker for Click actions.
